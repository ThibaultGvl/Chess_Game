# Projet POO2 : Jeu d'échecs et variant

L'application codée en Java permet de jouer aux échecs orthodoxes ainsi qu'au Grand Acedrex.

## Installation

Pour commencer, téléchargez le projet. Ensuite, assurez-vous d'avoir Java 17 ou une version plus récente sur votre machine. Si ce n'est pas le cas, vous pouvez la télécharger ici : https://www.oracle.com/java/technologies/downloads/#jdk17-windows. Maintenant, vous pouvez installer le projet de 3 manières différentes : en cliquant sur le fichier .jar qui se trouve dans le répertoire "jar", en exécutant la commande suivante depuis le terminal lorsque vous vous trouvez dans le dossier "jar" du projet : `java -jar Chess.jar`, ou en l'ouvrant dans un éditeur et en le lançant depuis celui-ci.

##### ATTENTION : Il se peut que l'affichage ne soit pas optimal selon la mise à l'échelle de votre écran. Nous ne pouvons pas garantir une expérience de jeu optimale pour une mise à l'échelle différente de 100%.

## Utilisation

Une fois le programme ouvert, vous avez accès à deux boutons vous permettant de jouer soit aux échecs orthodoxes, soit au variant moyen-âgeux : le Grand Acedrex. Faites votre choix et jouez ! Si vous ne connaissez pas les règles de chacun des échecs, les voici : https://ecole.apprendre-les-echecs.com/regles-echecs/. Si vous ne connaissez pas les règles du Grand Acedrex : http://history.chess.free.fr/acedrex.htm. En résumé, si vous connaissez les échecs orthodoxes, le Grand Acedrex suit sensiblement les mêmes règles : échec, échec et mat... Sauf que les pièces sont, pour certaines, différentes et ont des mouvements différents. Le roque n'existe pas dans ce variant.
